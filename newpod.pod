=pod

=head1 NAME

newpod - Create a new plain old documentation file from the shared template.

=head1 SYNOPSIS

C<newpod foo.pod>

=head1 DESCRIPTION

B<newpod> creates a new .pod (Plain Old Documentation) file from the shared template, using the ~/.pudrc file for copyright notices specific to each user.


=head1 FILES

~/.pudrc

/usr/share/pud/template/pod.template

/var/log/pud/error.log

=head1 SEE ALSO

L<perl>

L<newpm>

L<newpl>
